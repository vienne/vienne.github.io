nyc.food.violations

nyc.food.violations is an app that allows you to enter your address and locates nearby restaurants with information on the number and types of health code violations from the last year, as well as the number of Foursquare checkins. This app uses big data sets from NYC Open Data, Google Maps Javascript API and Foursquare's API.

NYSKI

NYSKI is an an online hub for skiers that provides the opportunity to find information on mountains in the Northeast as well as the ability to coordinate trips with fellow skiers. Feature functionality includes ability to join trips, obtain weather and snow report, and get directions to selected mountains. It uses Geokit to geocode user addresses, Google Maps API, Snow Report API, JavaScript, and AJAX.
